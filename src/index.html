About
TaskFlowy is a WorkFlowy clone that provides three main features different from WorkFlowy
  • Offline: It installs a service worker and works using IndexedDB as an offline first cache.
  • Signed Out: No need to sign in to anything to use it, works out of the box with the browser cache. The browser cache will generally persist unless it is unused for a very long time.
  • Moving Storage: To get off-browser persistence, for example to sync the same set of tasks on another device, allows plugging in your own storage server with an OAuth key, or using Firebase (a backend-as-a-service from Google) Firestore.
